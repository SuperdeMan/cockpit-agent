# docs/research/ — 外部对标与行业调研

放**对外部产品 / 行业的调研、对标、竞品分析**笔记。与 [`docs/design/`](../design/) 区分：
- `docs/design/`：**本项目自己**的设计与落地记录。
- `docs/research/`：**别人怎么做**的调研，给本项目决策做依据，不含本项目实现细节。

## 约定
- 文件名：`YYYY-MM-DD-<topic>.md`，按调研日期前缀。
- 每篇开头注明：调研日期、目的、信息边界（一手/二手/推断）。
- 引用外部信息**带可点 markdown 链接**；URL 不确切时给来源站点名，不编造深链。
- 时点对标，结论以发布日为准；更新时追加「更新」小节，不覆盖原结论。
